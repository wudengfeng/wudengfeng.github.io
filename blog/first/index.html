HuGo搭建个人博客

  1. 安装hugo

在Hugo Releases下载对应系统版本的二进制文件

添加环境变量或者链接，方便使用

   #Mac下安装(没试过)
   brew install hugo
  1. 生成站点
   #以windows为例，自定义站点路径D:\BlogServer\site\
   hugo new site D:\BlogServer\site\

   #目录结构
   archetypes/
   content/
   data/
   layouts/
   resources/
   static/
   themes/
   config.toml
  1. 创建文章
   #被创建到content/下
   hugo new hello.md
   
   #被创建到content/blog下
   hugo new blog/HuGo搭建个人博客.md

   #HuGo搭建个人博客.md有如下初始化内容
   #去掉draft: true或者改为false才可以在运行时不指定buildDrafts时展示出该文章
   title: "HuGo搭建个人博客"
   date: 2018-07-19T23:47:49+08:00
   draft: true
  1. 安装皮肤

到皮肤列表挑选好看的皮肤，下载到themes/中。多下几个慢慢挑

   #hugo-redlounge是我现在在用的一个皮肤
   cd themes
   git clone https://https://github.com/tmaiaroto/hugo-redlounge.git
   cd ..
  1. 配置皮肤等参数
   #vim config.toml
   #很关键的一条，theme必须要写，不然就panic
   
   baseURL = "http://localhost/"
   languageCode = "en-us"
   title = "爬山的博客"
   theme = "hugo-redlounge"
  1. 运行hugo
   #默认端口1313
   hugo server &
   #调试模式加参数 --buildDrafts，可以看到draft: true的文章
  1. 部署到github 【可选】
