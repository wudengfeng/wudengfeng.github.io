HuGo搭建个人博客

1.安装hugo

在Hugo Releases下载对应系统版本的二进制文件

添加环境变量或者链接，方便使用

#Mac下安装(没试过)
brew install hugo

2.生成站点

#以windows为例，自定义站点路径D:\BlogServer\site\
hugo new site D:\BlogServer\site\

#目录结构
archetypes/
content/
data/
layouts/
resources/
static/
themes/
config.toml

3.创建文章

#被创建到content/下
hugo new hello.md
   
#被创建到content/blog下
hugo new blog/HuGo搭建个人博客.md

#HuGo搭建个人博客.md有如下初始化内容
#去掉draft: true或者改为false才可以在运行时不指定buildDrafts时展示出该文章
title: "HuGo搭建个人博客"
date: 2018-07-19T23:47:49+08:00
draft: true

4.安装皮肤

到皮肤列表挑选好看的皮肤，下载到themes/中。多下几个慢慢挑

#purehugo是我现在在用的一个皮肤
#hugo-redlounge的左半边加上purehugo的右半边不错
cd themes
git clone https://github.com/dplesca/purehugo.git
cd ..

5.配置皮肤等参数

#vim config.toml
#很关键的一条，theme必须要写，不然就panic
   
baseURL = "https://wudengfeng.github.io/"
languageCode = "cn"
title = "爬山的博客"
theme = "purehugo"

6.运行hugo

#默认端口1313
hugo server &
#调试模式加参数 --buildDrafts，可以看到draft: true的文章

7.部署到github 【可选】

在github上创建一个仓库，命名格式[username].github.io（比如wudengfeng.github.io）

创建完这个仓库，在Setting->Github Pages->Theme Chooser->Change theme随便选一个主题，Select theme

#运行生成静态页面
hugo server --baseUrl="https://wudengfeng.github.io/"

静态页面生成到public目录下，将public目录下所有文件推送到master分支上

cd public
git init
git remote add origin https://github.com/wudengfeng/wudengfeng.github.io.git
git add -A
git commit -m "first commit"
git push -u origin master

浏览器访问https://wudengfeng.github.io/

参考Hugo中文文档
